# Projeto-Banco-de-Dados----Sistema-de-Ordem-de-servi-o-Urbana
O trabalho busca uma solução rápida para ordem de serviços urbanas, onde o próprio cidadão poderá solicitar uma ordem de serviço para algum problema identificado, além disso, o próprio cidadão poderá acompanhar como está o andamento do serviço solicitado.
